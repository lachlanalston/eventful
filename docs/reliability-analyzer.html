How to export from Reliability Monitor
  1. Press Win + R, type perfmon /rel, press Enter
  2. In the Action menu, click Save Reliability History…
  3. Save the .xml file anywhere
  4. Upload that file here

Alternatively: search Reliability History in the Start menu and use the Action menu from there.
