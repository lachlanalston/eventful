How to export from Windows Event Viewer
  1. Open Event Viewer — press Win + R, type eventvwr.msc, press Enter
  2. In the left panel, navigate to the log you want (e.g. Windows Logs → System)
  3. Click Action in the menu bar, then Save All Events As…
  4. In the "Save as type" dropdown, select XML (*.xml)
  5. Choose a filename and save — then upload that file here

No access to Event Viewer? Use the event log export script to generate the file remotely.
