How to generate ReliabilityHistory.xml
  1. Download and run Get-EventLogExport.ps1 on the target machine (as Administrator)
  2. The script creates a ZIP at C:\Temp\EventLogs_HOSTNAME_DATE.zip
  3. Extract the ZIP and locate ReliabilityHistory.xml
  4. Upload that file here

The script is available at tools.lrfa.dev
