  1. Download Get-EventLogExport.ps1 from tools.lrfa.dev
  2. Run it on the target machine: .\Get-EventLogExport.ps1
  3. It exports System, Application, Security, Setup logs + Reliability Monitor history to a ZIP
  4. Upload that ZIP here

Individual files also work — drop a single XML if you only have one log.
